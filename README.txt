this creates or modify's your .bash_profile script in your home directory
to point to the common bash_profile.sh in this project.

to begin open up a terminal, cd to where you cloned this project
and execute

./setup

thats it!


if changes are made by any team member to bash_profile.sh in the git repo,
just run:

git pull

and the new changes will be applied the next time you restart your terminal
